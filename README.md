# deadlines
Deadline tracker for Pi Class

The Problem
When you're working in a group, tasks have a way of disappearing into the void. Someone says they'll handle it, but there's no record of who, or when it's due. Deadlines get missed not because people are lazy, but because nothing was written down in a place everyone could see. Spreadsheets get messy. Group chats get buried. Tools like Asana are overkill when your team is 15 people trying to get something done.

What This Does
Deadline Tracker is a single HTML file you can open in any browser. No account, no server, no setup.
You can:
- Add tasks with a title, an assigned team member, a due date, and an optional due time
- See at a glance who owns what and when it's due
- Filter by overdue, due soon, or completed tasks
- Mark tasks complete or delete them when they're done
- See a live count of total, overdue, upcoming, and completed tasks at the top
Tasks are saved automatically in your browser so nothing is lost when you close the tab.

Who It's For
Any small group that needs a shared to-do list with deadlines and ownership. It was built for a 15-person team but works for any group size.

How To Use It
1. Get the link
2. Open it in any browser (Chrome, Safari, Firefox, Edge — all work)
3. Click Add task to create your first task
That's it. No login. No subscription. No tutorial needed.

Tech
One plain HTML file. No frameworks, no build tools, no dependencies. Uses browser localStorage to save tasks between sessions.
